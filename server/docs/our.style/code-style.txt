java:
0.设置ide(eclipse)的编码为uft8，使用默认格式化方案；
1.括号采用行尾风格；
2.tab和space设置采用ide默认方案；
3.尽量填写函数前javadoc注释；
4.变量名
    首字母小写，后面单词首字母大写
    不用匈牙利命名法，
    尽量选取合适的英文单词，
    不要过长，
    尽量不采用缩写，除了最常见的；
5.复合语句块都用{}括起来；
6.注释尽量用容易理解的英文，难以表达清楚的用中文；
7.类名每个单词首字母大写；
8.static final用以下划线分隔的全大写单词命名；


sql:
1.表名每个单词首字母大写；
2.表名不用复数词；
3.表名，字段名，变量名等
    尽量不采用缩写，除了最常见的，
    避免过长，
    避免保留字，
    尽量用英文，难以翻译的用拼音；
4.关键字全大写；
5.bool类型的字段名用肯定形式；
6.表名，字段名用PascalCase；
7.主键像表名+Id这样命名，其它字段不加表名作前缀，外键名和主键名一样；
8.数据库命全小写
